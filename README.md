## coupang-eats 프로젝트

    크기에 따른 반응형 웹을 이용한 쿠팡이츠 사이트를 구현했습니다.
    bootstrap과 CSS를 사용하여 화면을 만들었습니다.

#### 기술
    Front-End : HTML5, CSS3, JavaScript, JQuery, bootstrap
